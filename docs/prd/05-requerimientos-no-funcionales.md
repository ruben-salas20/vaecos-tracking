# 05. Requerimientos No Funcionales

## RNF-01 Disponibilidad

- Plataforma activa en nube para operacion continua.
- Recuperacion basica ante fallos de procesos de corrida.

## RNF-02 Rendimiento

- Ejecucion paralela configurable.
- UI responsiva con tablas grandes (filtros/paginacion).

## RNF-03 Seguridad

- Hash seguro de contrasenas.
- Sesiones seguras.
- Control de acceso por rol.
- Auditoria de acciones criticas.

## RNF-04 Observabilidad

- Logs estructurados por modulo y corrida.
- Metricas tecnicas y operativas para diagnostico.

## RNF-05 Escalabilidad

- Separacion de responsabilidades (web, motor, almacenamiento).
- Preparado para crecimiento de usuarios y carga.

## RNF-06 Calidad operativa

- Trazabilidad completa de cambios.
- Capacidad de reconstruir decisiones por regla y corrida.
