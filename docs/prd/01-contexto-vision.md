# 01. Contexto y Vision

## Contexto

VAECOS inicio como una automatizacion para sincronizar estados de guias entre transportistas y Notion. El MVP actual evoluciono a:

- Motor operativo `v0.2` con corridas, reglas y trazabilidad.
- Capa web `v0.3` con vistas operativas y analytics.
- Persistencia historica en SQLite.

El siguiente salto es convertirlo en una plataforma web operativa en nube, multiusuario y siempre activa.

## Vision

Construir una plataforma operativa interna de tracking para 4 operadores que permita:

- Ejecutar y monitorear corridas en tiempo real.
- Reducir trabajo manual en excepciones.
- Asegurar trazabilidad por guia, corrida y cliente.
- Consolidar analytics operativos para toma de decisiones.
- Reemplazar completamente Notion cuando el producto este estable.

## Tipo de producto objetivo

Plataforma operativa interna formal, con potencial de evolucionar a SaaS en el futuro.
